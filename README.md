## About The Project

To create a social media app, targeted towards the Professional Photographer.
Therefore, to create an environment where the users can use a different information-sharing style than the existing social media photo apps.
With photo metadata, create an easy, yet more informative approach for the image-sharing community.
User interaction between each other on like and comments.

## API
### User
#### GET Get A User

**Endpoint:** `GET /api/users/userId`

**Description:** Retrieve a user data.

**Example:**
curl -X GET https://https://picconnect-backend.onrender.com/api/users/QCRTMLLH6ZR8Tmi5KCBuQADMHg02

#### PUT Update A User

**Endpoint:** `PUT /api/users/userId`

**Description:** Update a user data.

**Body:** <br/>
<ul>
  <li>name: STRING</li>
  <li>profilePicUrl: STRING</li>
  <li>birthday: DATE</li>
  <li>phoneNumber: STRING</li>
  <li>about: STRING</li>
  <li>rating: INTEGER</li>
  <li>email: STRING</li>
</ul>

**Example:**
curl -X GET https://https://picconnect-backend.onrender.com/api/users/QCRTMLLH6ZR8Tmi5KCBuQADMHg02
